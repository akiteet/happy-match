# 开心消消乐 · HTML5 复刻

纯前端三消小游戏，零依赖，双击 `index.html` 即可在浏览器中游玩。

**仓库：** [github.com/akiteet/happy-match](https://github.com/akiteet/happy-match)

## 在线试玩（GitHub Pages）

**入口：** [https://akiteet.github.io/happy-match/](https://akiteet.github.io/happy-match/)

手机/电脑浏览器直接打开即可，无需安装。

## 如何运行

```bash
# 方式一：在线
# 打开 https://akiteet.github.io/happy-match/

# 方式二：本地直接打开
# 双击 index.html，或用浏览器打开该文件

# 方式三：本地静态服务（可选）
npx serve .
# 或 python -m http.server 8080
```

推荐使用 Chrome / Edge / Firefox 最新版。

## 操作

| 操作 | 说明 |
|------|------|
| 拖动 | 按住棋子向相邻方向滑动交换 |
| 点选 | 先点一颗，再点相邻一颗交换 |
| 提示 | 高亮一对可走步（每关有限次数） |
| 音效 | 开关 WebAudio 合成音效 |
| 重开 | 重新开始当前关卡 |

无效交换会回弹且**不扣步**；有效交换扣 **1** 步，连锁消除不扣步。

## 玩法规则

### 基础

- 8×8 棋盘，6 种动物：🐰 🐻 🐼 🐸 🐯 🐱
- 横或竖连续 **3 个及以上** 同色即可消除
- 消除后上方棋子下落，顶部补新棋子；可形成连锁

### 特殊糖果

| 形状 | 生成 | 效果 |
|------|------|------|
| 横/竖 4 连 | 直线糖 | 消除整行或整列 |
| L / T 形 | 炸弹 | 消除周围 3×3 |
| 横/竖 5 连 | 魔力鸟 ✨ | 与某色交换后消除全盘该色 |

特殊糖可与普通棋子或其它特殊糖组合激活（如双魔力鸟清盘等）。

### 障碍

| 障碍 | 说明 |
|------|------|
| 冰块 | 盖在棋子上，需多次命中（层数角标）；冰绑定格子，不随棋子乱飞 |
| 藤蔓 | 占格不可交换，相邻消除会扣血，归零后变成普通棋子 |
| 牢笼 | 锁住棋子不可交换，可被匹配/特效消除或相邻消除解锁 |

### 关卡

- 多关线性推进：分数目标 / 消除指定色 / 清冰等
- 步数耗尽未达成目标则失败，可重试
- 过关后进入下一关

## 界面说明

- **深色主题**：深紫棋盘衬托彩色 emoji
- **特殊糖果**：直线光带、炸弹双环、魔力鸟彩虹环
- **障碍**：冰罩层数、藤蔓叶片 + HP、金属牢笼栅栏

## 开发备注

- 棋盘逻辑坐标固定为 512×512，按 `devicePixelRatio` 缩放以保持清晰
- 状态机：`idle` → 交换动画 → 匹配/激活 → 消除动画 → 重力下落 → 连锁或回到 `idle`
- 冰块为**格子属性**：消除时减层，重力只移动棋子颜色/特殊类型
